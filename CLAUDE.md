# Atitlán Experience - Contexto del proyecto

## Stack
- Frontend: React + TypeScript + Vite
- Backend/DB: Supabase (PostgreSQL)
- Imágenes: Cloudinary
- AI: Gemini API
- Deploy: Vercel
- Estilos: Tailwind CSS

## Problemas conocidos
1. El sitio es muy lento en carga general
2. El formulario del back office para editar/agregar tours falla intermitentemente
3. Los nombres de archivos no se sincronizan correctamente con Cloudinary
4. Hay fallas recurrentes no identificadas
